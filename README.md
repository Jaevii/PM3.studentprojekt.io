## ğŸš€ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Jaevii/iBoxen-Website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd iBoxen-Website
   ```

3. Open `index.html` in your web browser or serve it using a local server:
